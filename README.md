## General info
This project is created for data visualization in Apache SuperSet application, using Flask framework and Postgres.
	
## Technologies
Project is created with:
* cryptography==2.4.2
* jsonschema==3.2.0
* matplotlib==3.3.3
* nltk==3.5
* pipenv==2020.8.13

	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
